# AgMaps

Welcome to AgMaps! This documentation refers to [AgMaps](https://WishartLab.github.io/agmaps/).

In summary, there are three ways to run AgMaps:
1. As a Web-Assembly, Client-Side application run in your browser, hosted at (github pages link)
2. As a PyShiny, Server-Side application run in your browser, hosted at (server link)
3. As a Python application from your computer. See [Running](https://github.com/WishartLab/heatmapper2/wiki/Running) for details.
