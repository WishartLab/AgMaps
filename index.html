GEOMAP
Display values for defined territories on a map using GeoJSON files. Have temporal data? Scrub through time points with a media player-style slider.

Supports .txt, .dat, .tsv, .tab, and .xlsx files, plus .geojson for maps.
GEOCOORDINATE
Plot coordinates with associated values onto a map. Have temporal data? Scrub through time points with a media player-style slider.

Supports .txt, .dat, .tsv, .tab, and .xlsx files.
